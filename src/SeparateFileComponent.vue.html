Here starts the fun

This section is in different component called SeparateFileComponent.
But it's not an ordinary component, no no no.

It is special.

It was build on-fly using vue-separate-files-webpack-loader.

If you look at source code of this example, you won't see any SeparateFileComponent.vue; you'll see files:

SingleFileComponent.vue.html
SingleFileComponent.vue.js

Actual SingleFileComponent.vue file, which is consumed by vue-loader, is never physically created.
It's prepared in memory and passed on-fly.
