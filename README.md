# Vue separate files Webpack loader example app

This example app demonstrates how to use [vue-separate-files-webpack-loader](https://github.com/NetCZ/vue-separate-files-webpack-loader).

## Installation

Clone this repository.

Then run command:

``` bash
npm install
```
or
``` bash
yarn install
```

## Starting the app

Run command:

``` bash
npm run dev
```
or
``` bash
yarn dev
```
